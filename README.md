# movie-tv-site-info-archive
该项目专注于电影与电视剧网站的基础信息整理，对公开可访问的影视站点进行系统化归档。仓库内容以站点层面信息为主，便于后续分析站点结构、更新节奏及内容覆盖方向。项目不包含任何影视资源文件，也不提供播放或下载功能，仅作为资料存档与信息参考。所有内容来源于公开网络环境，保持客观中立立场。
